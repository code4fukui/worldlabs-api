# worldlabs-api

worldlabs-api is an API interface for [WorldLabs](https://www.worldlabs.ai/)

## setup

.env
```
WLT_APIKEY=***
```

## usage

```js
import { WorldLabs } from "./WorldLabs.js";

await WorldLabs.createModel("A warp hole suddenly appeared right before my eyes.");
```
